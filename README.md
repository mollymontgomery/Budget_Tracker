# Budget Tracker
Using a Progressive Web Application this application allows the user to add expenses and add to their budget with or without online connection.

1. npm i
2. npm i express
3. npm i mongoose

  2. CSS
  3. jQuery
  4. Express
  5. MongoDB
  6. Mongoose
  7. IndexedDB
